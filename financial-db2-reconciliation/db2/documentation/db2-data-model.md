# DB2 Data Model

<!-- Purpose -->
The DB2 layer stores master accounts, definitive movements, staging rows, control records, discrepancy rows and audit information.

<!-- Tables -->
The main tables are FIN_ACCOUNT, FIN_MOVEMENT, FIN_STG_MOVEMENT, FIN_RECON_CONTROL, FIN_DISCREPANCY and FIN_AUDIT_LOG.

<!-- Traceability -->
The model supports control, reconciliation and audit needs in a banking environment.
