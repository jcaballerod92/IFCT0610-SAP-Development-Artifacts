# Execution Guide

<!-- Step 1 -->
Compile the COBOL programs using COMPILE.jcl.

<!-- Step 2 -->
Run FINBATCH.jcl to execute validation, reconciliation and reporting.

<!-- Step 3 -->
Review the outputs in the control, discrepancy and report files.

<!-- Note -->
The repository is a simulation of a mainframe banking flow, so the execution artifacts are represented in a portable format.
