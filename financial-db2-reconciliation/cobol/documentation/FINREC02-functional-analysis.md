# FINREC02 Functional Analysis

<!-- Purpose -->
FINREC02 reconciles staged movements against reference data and definitive movements.

<!-- Input -->
The program reads the staging file generated by FINVLD01.

<!-- Output -->
It writes a reconciliation control file and a discrepancy file, and updates the staging status.

<!-- Control -->
The process reflects a typical financial batch reconciliation flow with clear success, difference and error outcomes.
